# How to use
This folder is meant to be cloned to gather local references to input data. Use the Rmarkdown to re-import the data and re-create the figures used in the publication. This Rmd document contains the commands to create Figure 3f and Figure 3g. 

The .Rda object found within the data/ folder is the log2 normalized Nanostring abundance within an Expression Set object along with the sample metadata. For the complete dataset, see GEO here: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE98291
