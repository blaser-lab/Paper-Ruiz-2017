# How to use
This folder only contains the 16S OTU analysis used to plot Figure 4d, Figure 4e and Figure 4f. The IPython notebook can be used as long as QIIME 1.9.1 scripts are installed and available in the $PATH. 

The files located in the data/ folder is the OTU table from the PulsePAT experiment. For the entire dataset, see the QIITA repository here: https://qiita.ucsd.edu/study/description/10527

Figure 4d, Figure 4f can be found within the IPython Notebook
Figure 4e can be found within the Rmarkdown document
